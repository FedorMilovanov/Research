# NEXT_CHAT_PROMPT — промт для следующего чата

У меня есть ZIP-пакет исследований по серии «Тёмная сторона кафедры». Нужно продолжить работу строго по нему.

Главные правила:

1. Не делать список плохих имён. Работать через богословские категории.
2. Библейские кейсы дают нормативные категории; исторические и современные кейсы только иллюстрируют.
3. Не плодить сущности: “волк” и “лжеучитель” часто один разрушительный комплекс с разных сторон.
4. По-русски не писать “диотрефический”; писать “служитель типа Диотрефа”, “Диотреф в церкви”, “человек, любящий первенство”.
5. Проверять каждый кейс по источниковой лестнице A/B/C/D.
6. Не смягчать судебно признанное преступление языком “moral failure”.
7. Не усиливать allegation до “доказано”.
8. Не путать прощение брата с восстановлением к кафедре.
9. Искать первоисточники, церковные statements, независимые reports, судебные документы и серьёзную журналистику.
10. Стиль сайта: русский, баптистско-реформатский, библейский, пастырский, академически честный, без истерики и клеветы.

Начать с файла `00_START_HERE/README_START_HERE.md`, потом `FINAL_TAXONOMY.md`, `CASE_VERIFICATION_REGISTER.md`, `SOURCE_PROTOCOL.md`, затем уже читать нужные v0–v3 файлы.
