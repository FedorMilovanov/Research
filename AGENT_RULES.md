# Правила для Агентов по сбору данных (Research Repo)

## 1. Иерархия Доверия к Источникам (Levels)
- **Level A**: Оригиналы документов, официальные заявления, судебные решения.
- **Level B**: Респектабельные СМИ, подтвержденные биографии.
- **Level C**: Блоги, соцсети, слухи.
- **HOLD**: Данные найдены, первичка не подтверждена.

## 2. Правила Структуры (Борьба с фрагментацией)
**ЗАПРЕЩЕНО** плодить мелкие файлы. 
**ПРАВИЛО**: Один кейс = Один большой файл-досье.

### Стандарт структуры досье:
1. `# [Имя Лидера]`
2. `## 🚩 Суть кейса`
3. `## 📄 Первичные Источники`
4. `## 🗣 Реакции и Ответы`
5. `## 🔍 Анализ и Верификация`
6. `## ⏳ Хронология`

## 3. Процесс обновления
Агент должен: Прочитать текущий файл $\rightarrow$ Интегрировать новые данные в нужные секции $\rightarrow$ Обновить файл целиком.
